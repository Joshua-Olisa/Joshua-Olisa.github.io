Projects

Natural Language Text Processing Analysis Airline Review

An end-to-end NLP pipeline that web-scrapes and cleans airline reviews, applies tokenization and lemmatization, then generates word clouds and frequency analyses. It uses VADER for sentiment scoring and Gensim LDA for topic modeling to deliver visuals and actionable insights.

Predictive Modeling of Airline Customer Bookings Using Ensemble and Gradient-Boosting ML Algorithms

The aim was to produce a robust predictive model that can identify booking events (the positive class) with strong recall while balancing precision to reduce false positives, demonstrate preprocessing strategies and modelling techniques suitable for imbalanced classification problems and to provide actionable insights and recommendations that could inform a production-ready model or a business decision-making process.

Customer Segmentation Using K-Means Clustering

The main goal is to identify the most important shopping groups based on income, age, spending score and provide recommendations to the marketing team to help develop a tailored strategy for different customer groups

Bank Customer Churn Analysis Insights and Strategic Recommendations

This project performs an exploratory analysis on a bank’s customer dataset to reveal the main drivers of attrition. By examining demographics, account balances, satisfaction scores, and regional trends, it uncovers patterns and provides data-backed recommendations for improving customer retention.

Customer Behavior Analysis: Data Driven Retail Decisions Store Performance

A full-cycle retail analytics project that ingests customer transaction data to uncover purchasing behaviors, identify sales drivers, and visualize performance trends. It defines control store metrics to evaluate trial layouts, leveraging Python and SQL for deep analysis and Tableau for interactive dashboards.

BellaBeat Data Analysis Capstone Project

This project uses Fitbit fitness tracker data to explore how users engage with their devices. Conducted as part of the Google Data Analytics Capstone, it applies SQL for data preparation and Tableau for visualization to identify patterns that can inform Bellabeat’s marketing strategy.
