# Portfolio Website
## Description
A professional, responsive portfolio website featuring dynamic UI elements like typewriter animations and a project carousel to showcase my data science and engineering expertise.


This project serves as a comprehensive digital resume and project hub, designed to present my technical skills in Python, SQL, and Tableau alongside my background as an Electrical/Electronics Engineer. Built with a focus on user experience, the site includes a dedicated "About" section detailing my professional journey , an interactive projects gallery that links directly to my GitHub repositories , and a functional contact modal that integrates with my email.

## Features

+ Dynamic Typewriter Animation: Automatically cycles through professional titles including "Electrical Engineer," "Data Scientist," "Data Analyst," and "Freelancer" on the home page.

+ Responsive Project Carousel: An interactive, touch-friendly showcase for my data analytics projects, featuring automated navigation indicators and responsive resizing for mobile devices.

+ Interactive Contact Modal: A custom-built pop-up form that allows users to send messages directly via an integrated mailto link.

+ Scroll-Triggered Animations: Includes a professional "About" section with images that animate into view as the user scrolls down the page.

+ Modern Navigation: Features a glassmorphism-style fixed navigation bar with a mobile-responsive "hamburger" menu toggle.

+ Project Showcase: Highlights major work such as my "Natural Language Text Processing Analysis" and "Bank Customer Churn Analysis," complete with descriptions and direct links.

+ Projects page with:
Same navigation bar, contact modal functionality and footer, Headers and description and Grid layout showing all 6 projects (2 per row on desktop, 1 per row on mobile) with clickable links to their respective GitHub repositories

## Built With
The website is built using a clean, "vanilla" technology stack for high performance and complete design control:

+ HTML5: Used for semantic structuring, meta-tag optimization for mobile responsiveness, and project content management.
+ CSS3: Utilized for complex layouts using Flexbox and Grid, custom linear-gradient styling, and interactive hover effects.
+ JavaScript (ES6+): Powers all site interactivity, including the carousel logic, typewriter effects, and modal state management.
+ Boxicons: Integrated for professional, scalable iconography across the navigation and social sections.
+ Google Fonts (Poppins): Applied globally for a clean, modern typography aesthetic.
